Simple click:
```
Return from (-99|68) to ‭(‭−‭110‬‬|‭60‬)‬ arrives at 13:28:21
```

Click with **CTRL** pressed:
```
Return from [x|y](-99|68)[/x|y] to [x|y]‭(‭−‭110‬‬|‭60‬)‬[/x|y] arrives at [b]13:28:21[/b]
```
